# MagicLand

Sitio web para la compra de tickets del parque de atracciones MagicLand. Permite a los usuarios explorar las atracciones disponibles, seleccionar fechas de visita, elegir el tipo de entrada y completar su compra de forma segura en linea.

## Herramientas

- HTML
- CSS
- JavaScript
- React
- Vite
- TailwindCSS
- React Router

## Instalacion

Instalar dependencias:

```
npm install
```

Iniciar servidor de desarrollo:

```
npm run dev
```

Generar build de produccion:

```
npm run build
```
